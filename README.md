# 🎓 OptiTalk — تعلّم الإنجليزية بالمحادثة الحية

<div dir="rtl">

تطبيق ويب لتعلم اللغة الإنجليزية عبر محادثة حية مع مدرس AI شخصي. الطالب يختار مدرسه من بين 6 شخصيات مختلفة، ويتحدث بصوته، ويردّ المدرس فوراً مع تصحيح الأخطاء وترجمة الكلمات الصعبة.

## ✨ المميزات

- 🎓 **6 مدرسين بشخصيات مختلفة** — صبور، مرحة، أكاديمي، تحفيزي، احترافي، إلخ.
- 🗣️ **محادثة صوتية حية** — استخدم الميكروفون للتكلم مع المدرس
- 📝 **أو اكتب بدلاً من ذلك** — وضع الكتابة متاح دائماً
- 🌐 **دعم العربية** — لو اتكلمت عربي، المدرس بيفهم ويرد بالإنجليزي ويصحّحلك
- 👨‍🏫 **مدرس بصورة حقيقية** — صور احترافية مولّدة بـ AI (بورتريه عمودي، خلفية مكتبية)
- 🎬 **أنيميشن طبيعي** — تنفس، رمش، حركة رأس، شفايف تتحرك لما بيتكلم
- 📹 **كاميرا الطالب** — شوف نفسك وأنت بتتكلم (اختياري)
- 🏆 **نظام نقاط وإنجازات** — حفّز نفسك على الاستمرار
- 🔥 **Streak يومي** — حافظ على سلسلة تعلمك
- 🎯 **تصحيح فوري** — أخطاؤك تتصلح لحظياً مع شرح بالعربي
- 💎 **ترجمة الكلمات الصعبة** — ترجمة فورية للكلمات الجديدة

## 🎨 التصميم

- تقسيم الشاشة: **المدرس فوق (35%)** → **كاميرا الطالب تحت (35%)** → **التحكم في النص (30%)**
- ثيم داكن أنيق مع لمسات ذهبية وبنفسجية
- واجهة عربية بالكامل (RTL)
- متجاوب مع جميع أحجام الشاشات

## 🛠️ التقنيات المستخدمة

- **Next.js 16** + **TypeScript** + **Tailwind CSS 4**
- **shadcn/ui** لمكونات الواجهة
- **Zustand** لإدارة الحالة
- **Framer Motion** للحركات والأنيميشن
- **Prisma ORM** + **SQLite** لقاعدة البيانات
- **z-ai-web-dev-sdk** لمحرك الـ AI
- **Web Speech API** للتعرف على الصوت والنطق
- **WebRTC** للكاميرا

## 🚀 التشغيل المحلي

```bash
# 1. تثبيت الاعتمادات
npm install

# 2. إعداد قاعدة البيانات
npx prisma db push

# 3. تشغيل خادم التطوير
npm run dev
```

ثم افتح [http://localhost:3000](http://localhost:3000)

## 📦 البناء للإنتاج

```bash
npm run build
npm run start
```

## 📱 التحويل إلى APK (TWA)

يمكن تحويل التطبيق إلى APK باستخدام Trusted Web Activity (TWA) عبر Bubblewrap أو PWABuilder:

```bash
# باستخدام Bubblewrap
npx @bubblewrap/cli init --manifest=https://your-domain.com/manifest.json
npx @bubblewrap/cli build
```
